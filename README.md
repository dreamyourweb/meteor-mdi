# mdi

Meteor package for Material Design icons from Templarian (http://materialdesignicons.com/)

## Installation

```bash
meteor add cedla:mdi
```

## Usage

Use the syntax below in your HTML files:
```html
<i class="mdi mdi-bell"></i>  <!-- bell -->
```
